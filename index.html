    What is JQRebound ?

    JQRebound is a jquery plugin allows you to bounce an element on a page in a simple and customizable manner . I especially developed for icons or logos but you can assign the script to any element with a small line of code.

    Usage.

    To use JQRebound, first we need to load the jQuery library in our page :

    <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7/jquery.min.js"></script>

    You need to download and load "easing" in your page too

    <script type="text/javascript" src="easing.js"></script>

    Now, download JQRebound, place it in the folder of your site and load it at the beginning of your html page like easing:

    <script type="text/javascript" src="JQRebound.js"></script>

    So JQRebound is loaded, now we need to call the function and parameterize the option if necessary. For this we proceed like this:

    <script>$(".rebound").JQRebound ()</script>          //with options
<script>$(".rebound").JQRebound ({options})</script>        //without options

    The customizable options are :

    begin,up,speed,replay and pos.

    the default settings are :

    begin   =0,
up  =50,
speed   =800,
replay  =true

    To change the settings value, we need to redefine the variables at the function calling. Like this :

    <script>$(".rebound").JQRebound ({replay:false})</script>

    Note: if I want to redefine many settings I have to separate them by commas except the last. like this :

    <script>$(".rebound").JQRebound ({replay:false,speed:850,up:60})</script>
